# **App Name**: Grace Hub

## Core Features:

- Member Directory: Display a searchable, sortable directory of church members.
- Group/Ministry Team sign-up: Allow members to request to join groups or ministry teams. The contact info of the team leader is displayed.
- Events Calendar: List upcoming events with details (date, time, location, description).
- Resources Section: Provide resources like articles, devotionals, or announcements to keep the congregation informed and engaged.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to evoke trust and serenity.
- Background color: A light, desaturated blue (#E3F2FD) to maintain a clean and airy feel.
- Accent color: A warm yellow (#FFCA28) to highlight important interactive elements, such as buttons or links.
- Body and headline font: 'PT Sans' (sans-serif) for readability and a modern feel.
- Use simple, clear icons from React Icons to represent different sections and actions.
- Employ a clean, grid-based layout with consistent spacing for a professional and organized appearance.
